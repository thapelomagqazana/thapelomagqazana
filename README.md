## 🧱 Core Projects

### 🧪 BrikByteOS — *Enterprise Release Intelligence Infrastructure*

> One CLI. One Schema. Enterprise Confidence.

**Category:** Unified Quality Orchestration (UQO)

**What it does:**
- Aggregates signals from:
  - Unit tests (**Jest**)  
  - UI tests (**Playwright**)  
  - Performance (**k6**)  
  - Security (**Trivy**)  

- Produces:
  - 📊 **Release Confidence Score (0–100)**  
  - 🚦 **Policy-based Gate** → APPROVED / REJECTED  

**Architecture:**
- Go (CLI-first, deterministic execution)
- Strict JSON schema normalization
- Policy engine (OPA-based)
- Reproducible, artifact-driven runs

```bash
bb run --all
bb score
bb gate --policy production
```
